# 운세 앱 (Unse App)

생년월일시로 한국 전통 명리를 계산하고, 분야별(취업·시험·연애·사업) 운세를 시기 흐름과 함께 AI로 풀어주는 완성된 단일 HTML 웹앱이다. 이 스킬의 핵심 산출물은 이미 `assets/` 폴더에 완성되어 있으므로, **새로 코드를 작성하지 말고 자산 파일을 그대로 사용자에게 제공하는 것**이 기본 동작이다.

## 무엇을 하는 앱인가

- 만세력 계산: 사주팔자 8글자(연·월·일·시주)를 천간지지로 계산. 절기(24절기) 기준 월주, 입춘 기준 연주, 진태양시 보정 시주.
- 오행 분포, 글자별 십신, 일간 강약
- 분야별 운세: 취업·직장운, 시험·합격운, 연애운, 사업운 등 사용자가 체크한 분야만 풀이
- 시기 흐름: 대운(10년)·세운(연)·월운(월) 표 + 시기별 운세 문장
- 음력 입력 → 양력 변환 (KASI 데이터 기반 라이브러리 내장, 1000~2050년)
- 시간 보정: 경도(서울 기준 약 -32분), 한국 표준시 변경 이력, 서머타임
- AI 풀이: 고른 분야별로 요약·시기별 흐름·조언을 JSON 구조로 생성해 접이식 카드로 표시
- 결과를 PDF로 저장(임베드 한글 폰트 기반 jsPDF + html2canvas)

## 두 가지 버전

`assets/` 폴더에 두 개의 완성 파일이 있다. 사용자의 사용 환경에 따라 적절한 것을 제공한다.

1. **`assets/unse-app.html`** — Claude 환경(claude.ai 아티팩트)에서 쓰는 버전.
   AI 풀이가 `api.anthropic.com`을 키 없이 호출하며, 이는 Claude 환경이 대행하므로 그대로 작동한다.
   **사용자가 이 앱을 Claude 안에서 쓰려는 경우(기본)** 이 파일을 사용한다.

2. **`assets/unse-app-local.html`** — 본인 컴퓨터에서 독립 실행하는 버전.
   AI 풀이가 사용자 본인의 Anthropic API 키로 호출된다(화면에서 키 입력, 브라우저 메모리에만 저장).
   **사용자가 파일을 다운로드해 로컬에서 쓰려는 경우** 이 파일을 사용한다.

## 작업 순서

1. 사용자가 어느 환경에서 쓸지 불명확하면 한 번 확인한다(Claude 안에서 / 내 컴퓨터에서).
2. 해당 버전의 HTML을 `/mnt/user-data/outputs/`로 복사한다. 예:
   ```bash
   cp assets/unse-app.html /mnt/user-data/outputs/unse.html
   ```
3. `present_files`로 사용자에게 전달한다. Claude 환경용(`unse-app.html`)은 아티팩트로 열리면 바로 작동한다.
4. 코드를 임의로 다시 작성하지 않는다. 사용자가 수정/기능 추가를 요청할 때만, 해당 HTML을 열어 필요한 부분을 편집한다.

## 중요한 주의사항

- **계산 로직은 사주 앱과 동일하게 검증되었다.** 천간지지 인덱스, 절기 함수, 오자둔/오호둔, 십신 본기 음양, 대운 남순여역 등 계산부를 함부로 바꾸지 말 것. 수정 요청 시에도 계산부는 신중히 다루고, 바꾼 경우 알려진 사주로 재검증한다.
- **로컬 버전에 API 키를 하드코딩하지 말 것.** 반드시 화면 입력 방식을 유지한다(로컬 전용, 웹 배포용 아님).
- **운세 풀이는 참고용**이다. AI 풀이 프롬프트는 단정적 예언·점집식 위협을 금지하고, 따뜻한 존댓말로 자기 이해를 돕는 톤을 유지하도록 되어 있다. 이 방침을 약화시키지 말 것.
- 음력은 양력 1000~2050년 범위에서만 정확하다(내장 라이브러리 한계).
- 절기는 천문 근사식이라 절입 경계일에 드물게 하루 오차가 날 수 있다.

## 흔한 수정 요청과 위치

- 풀이 말투/타깃 변경 → HTML 내 `const prompt =` 블록
- 풀이 분야(취업·연애·사업 등) 추가/변경 → 입력 화면의 `.chip` 체크박스 + 프롬프트의 분야 안내 + `renderInterp`
- 풀이 항목(카드) 구조 → 프롬프트의 JSON 형식 + `parseInterp` / `renderInterp` 함수
- 디자인/색상 → `<style>` 내 CSS 변수(`:root`)와 섹션 스타일
- PDF 출력 모양 → `savePDF()` 함수(jsPDF). 대운·세운은 한 페이지, 월운은 다음 페이지에 큰 카드로 배치되며, 카드 높이는 `luck(...)` 호출의 마지막 인자(chOverride)로 조정한다.

## PDF 레이아웃 메모

- `savePDF()`가 메인(임베드 한글 폰트), `savePDF_imageFallback()`가 폰트 로드 실패 시 이미지 방식.
- 운(運) 표는 `luck(num, title, trackSel, noteSel, gapAfter, chOverride)`로 그린다.
  - 대운 + 세운: 한 페이지(`chOverride=26`)
  - 월운: 다음 페이지 단독, 카드를 세로로 키워 채움(`chOverride=50`)
  - 칸 높이가 40 이상이면 칸 안 글씨/줄간격이 자동으로 커진다(`const big = ch>=40`).
